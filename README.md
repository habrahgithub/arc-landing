# ARC Landing

Static landing site for ARC, served at:

- `https://arc.starwealthdynamics.org/`

ARC is the **Audit Layer for AI-Assisted Coding**.

## Deploy

This directory is linked to the Vercel project `arc-landing` via `.vercel/project.json` (local metadata; not committed).

From this folder:

```bash
vercel --prod
```
